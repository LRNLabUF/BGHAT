If using this template for publication, please cite (found in this folder):

Prodoehl, J., Yu, H., Little, D.M., Abraham, I., Vaillancourt, D.E., 2008.  Region of interest template for the human basal ganglia: comparing EPI and standardized space approached. Neuroimage 39, 956-965.

This folder contains a total of 11 NIfTI files.  The below 10 files are binarized and contain a single region.  Voxels included in the regions are represented with a value of 1.

BGHAT_Left_Caudate.nii
BGHAT_Left_GPe.nii
BGHAT_Left_GPi.nii
BGHAT_Left_Putamen.nii
BGHAT_Left_STN.nii
BGHAT_Right_Caudate.nii
BGHAT_Right_GPe.nii
BGHAT_Right_GPi.nii
BGHAT_Right_Putamen.nii
BGHAT_Right_STN.nii

Additionally, a single file (BGHAT.nii) contains all 10 regions of the BGHAT in the 1mm MNI space.  We have used values that range from 1-10, with the value of each voxel corresponding to a specific region of the BGHAT (shown in the below key).

KEY
Right_Caudate	1
Right_Putamen	2
Right_GPe	3
Right_GPi	4
Right_STN	5
Left_Caudate	6
Left_Putamen	7
Left_GPe	8
Left_GPi	9
Left_STN	10
